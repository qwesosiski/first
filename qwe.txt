Клонирование в Git Bash — это копирование удалённого репозитория на локальный компьютер. При этом переносятся все файлы и папки проекта, а также вся его история с момента создания.
Репозиторий в Git Bash — это папка проекта, отслеживаемая Git, содержащая дерево изменений проекта в хронологическом порядке.
«Запушить» в Git — это отправка последних фиксаций с локального репозитория в удалённый.
